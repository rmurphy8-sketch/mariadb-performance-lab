# MariaDB Performance Lab 🚀
This project explores database performance on **ARM-based (aarch64)** architecture using a **MediaTek MT8186** processor.

## Project Scope
- Analysis of connection overhead vs. batch transaction throughput.
- Documentation of MariaDB 10.11 setup on ChromeOS Linux (Debian 12).

## Key Results
- **Individual Inserts:** ~10/sec (High latency).
- **Batch Transactions:** ~50+/sec (Optimized throughput).

## Environment
- Hardware: "Magneton" (Lenovo IdeaPad Slim 3)
- OS: Debian 12 (Bookworm) inside ChromeOS
